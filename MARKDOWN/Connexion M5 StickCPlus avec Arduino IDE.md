# Connexion du M5 StickC Plus avec le logiciel Arduino IDE

Pour connecter le M5 StickC au logiciel Arduino IDE, il faut les éléments suivants :
- Logiciel Arduino IDE
- Un M5 StickC Plus
- Un câble USB-A vers  USB-C
